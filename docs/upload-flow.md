# Upload Flow

## Current Flow

ProfileScheduler
→ ProfileWorker
→ Generate content
→ Export MP4
→ Upload to YouTube
→ Upload to social platforms
→ Save upload history
→ Cleanup

## Upload Platforms

* YouTube
* Facebook
* Instagram
* TikTok
