# Video Pipeline

## Image Mode

* Scan images
* Create SlideItems
* Apply filters
* Apply Ken Burns
* Render frames
* Encode H264
* Add AAC audio

## Video Mode

* Scan root/video folder
* Create VideoClipItems
* Use Media3 Transformer
* Merge clips
* Export MP4

## Output

* 1080x1920
* 30fps
* H264 MP4
